국토지리정보원 공간정보 메인화면
① 국토지리정보원 로고 : 국토지리정보원 홈페이지로 이동한다.
② 상단메뉴 : 사용자 표시, 로그아웃, 정보수정 및 홈페이지로 이동한다.
③
시스템메뉴 : 지형지물 변동 을 클릭하면 우측에 지형지물 변동사이트 핫링크 화면이 나타난다.
④
상단의 서브매뉴 : 왼쪽에서 4번째 을 클릭하면 지형지물 변동 지도화면으로 전환된다. (7페이지 참조)
⑤
지도변경신고 : 지도변경신고 를 클릭 하면 지도변경 신고 기능으로 전환된다.(11 페이지 참조)
  ＊ 기관권한사용자는 기관신고화면으로 전환되며, 일반사용자는 일반신고화면으로 전환
⑥
지도변경조회 : 지도변경조회 를 클릭하면 지도변경조회 기능이 활성화 된다.(13 페이지 참조)
⑦
지형변화정보 : 지형변화정보 를 클릭하면 지형변화정보 자료공개 화면으로 전환된다.
