지형지물변동 > 지도화면 설명
①
버튼을 클릭하면 주소/명칭 위치검색 창 펼치기/접기를 한다.
②
에 따른 위치정보를 [검색] 버튼을 클릭하여 검색하여 위치 목록을 조회한다.
③
검색결과 목록에서 해당 항목을 클릭하면 해당위치로 이동하며, 지도위치 이동 시
축척은 1/10000으로 이동한다. (9페이지 참조)
 
④
을 클릭하면 이전, 처음 페이지네이션된 위치검색 목록으로 전환된다.
⑤ 번호를 클릭하면 페이지네이션된 위치검색 목록의 해당 페이지 목록으로 전환된다.
⑥
을 클릭하면 다음/최종 페이지네이션된 위치검색 목록으로 전환된다.
⑦
지도컨트롤버튼(확대/축소/전체보기/이동/이전보기/이후보기/거리측정/
삭제하기)
 
⑧
버튼을 클릭하여 범례 보기/감추기를 한다.
⑨ 범례종류를 나타내며 체크버튼을 사용하여 심볼에 대한 보이기/감추기를 한다. (10페이지 참조)
⑩ 지도선택 창에서 선택한 지도와 인덱스가 배경지도에 표현된다. (11페이지 참조)
⑪ [지도선택] 버튼을 클릭하면, 지도선택 창 보이기/감추기를 한다.
⑫ 색인창으로 배경지도의 영역을 표현한다.
⑬ 배경지도를 미리 정해진 축척으로 확대/축소한다.
