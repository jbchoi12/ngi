국토지리정보원 홈페이지 로그인 후 공간정보 접속
① 국토지리정보원 홈페이지(http://www.ngii.go.kr)로 접속하여 사용자 로그인을 한다.
② 홈페이지 상단의 [공간정보] 메뉴를 클릭하면 공간정보 페이지로 이동한다.
  ※ 공간정보 페이지는 http://210.117.198.32:9000 으로 직접 접속 가능(7페이지 참조)
③ 공간정보 페이지 우측상단의 아이콘을 클릭하면 국토지리정보원 홈페이지로 이동한다.
④
공간정보 페이지에서 좌측 지형지물변동 클릭 후 우측의 지도변경신고 를 클릭하여
지도화면으로 전환한다.
⑤
공간정보 페이지 우측상단 4번째 지형지물변동 을 클릭하여 지도화면으로 전환한다.
  ＊ 지도화면은 http://210.117.198.32:9000/sub/map.jsp?serv_cd=4로 직접 접속 가능(8페이지 참조)
⑥ 국토지리정보원 홈페이지에서 지형지물변동신고 [지형지물 변동 신고] 버튼을 클릭하여 지도화면으로
바로 전환이 가능하다. (8페이지 참조)
