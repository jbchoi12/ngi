지형지물변동 > 지도변경신고 > 기관신고
① 지도변경신고 메뉴에서 [지도변경신고] 메뉴를 클릭하면 지도변경보고조회 창이 나타난다.
② 지도변경정보 조회조건을 설정한다.
③ [조회] 버튼을 클릭하면 조회결과가 조회목록 화면에 나타난다.
 
- 변동유형과 공사명, 보고월, 대상지역, 상태, 이 조회된다.
 
- 첨부파일 등록상태, : 배경지도에 해당공사 위치를 직접등록 및 이동
 
※ 첨부파일이 없는 경우 회색으로 표시되며, 등록된 경우 색상이 나타난다.( )
  회색 버튼을 클릭하여 해당 공사의 첨부파일을 등록할 수 있다.
④ 조회목록 중 해당항목(공사)을 클릭 시 상세보기 화면이 나타난다.(14페이지 참조)
⑤
첨부파일 버튼 클릭 시 착공/준공/위치도 첨부파일 업로드 화면이 나타나다.
⑥
을 클릭하면 배경지도에 해당공사 위치를 직접 등록하거나 등록된 위치로 이동한다.(13페이지 참조)
⑦ [엑셀저장] 을 클릭하면 조회목록 중 체크 선택한 목록에 대하여 엑셀파일로 저장한다.
  ※ 저장한 엑셀파일은 다음 등록 시 추가작성(신규작성, 변경작성)하여 [신규등록] [엑셀저장]을
통해 등록할 수 있다.(17페이지 참조)
⑧ [신규등록] 버튼을 클릭하면 지도변경보고등록 창이 나타난다.(15페이지 참조)
⑨ [닫기] 버튼을 클릭하면 신고기능을 종료한다.
