지형지물변동 > 지도변경조회(기관신고) > 상세보기
① 지도변경조회(기관신고)창의 조회결과 목록에서 공사정보를 클릭하면 기관신고 상세화면창이 나타난다.
② 해당 변경신고 신청한 사용자의 정보가 나타난다. (해당 기관사용자 조회 시)
  ※ 다른 기관 사용자가 조회 시 기관명만 보이게 된다.
③ 변동정보 상세 내용을 열람한다.
④
지도화면에서 해당 위치로 이동 또는 위치 입력(해당 기관 사용자만 가능 13페이지 참조)
⑤ 변동정보의 처리 사항을 나타낸다.
  1. 접수중 – 변동정보 신고된 상태
  2. 보완필요 – 신고된 변동정보가 내용이 부족하여 보완 요청된 상태
  3. 접수완료 – 변동정보의 내용을 확인하고 첨부파일 첨부가 완료된 상태
  4. 지도수정중 – 지도수정 업체로 작업 지시가 된 상태
  5. 지도수정완료 – 지도수정된 성과를 확인하여 검수가 완료된 상태
  6. 고시완료 – 지형고시를 완료하여 국토지리정보원에서 수정된 지도가 판매되는 상태
⑥ 기관신고-상세화면 창을 종료하고 지도변경 조회 창으로 돌아간다.(18페이지 참조)
