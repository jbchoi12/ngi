※ 제품 설치 후 계속적으로 설치페이지 접근 시 점검사항
1. IE 10/11 사용자의 경우 IE 메뉴 > 도구 > 호환성 보기 설정
2. 웹사이트 추가 후 페이지 재 접속
3. 이외의 브라우저에서 설치 시 설치파일을 다운로드 후 다운로드 위치로 이동
4. 다운로드 된 설치파일 마우스 오른쪽 버튼 클릭 > 관리자 권한으로 실행
5. 이외의 사항으로 설치가 완료되지 않는 경우 관리자에게 문의하시기 바랍니다.
※ 설치파일 실행 시 액세스 거부 오류가 발생하는 경우
1. 설치파일 다운로드 위치로 이동
2. 설치파일을 선택 후 마우스 오른쪽 버튼을 클릭하여 [속성] 선택
3. [일반] 탭에서 차단 설정이 되어있다면, [차단 해제] 버튼 클릭 후 적용
4. 설치 파일 실행
